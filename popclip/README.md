# PopClip 插件

macOS PopClip 插件，选中文字后快速保存到 Heptabase。

## 前提条件

需要先安装并配置 Python 后端，详见 [../python-backend/README.md](../python-backend/README.md)。

## 安装

双击对应的 `.popclipext` 文件夹即可安装：

- **Heptabase Journal.popclipext** — 追加选中文字到今日日志
- **Heptabase Card.popclipext** — 保存选中文字为笔记卡片（第一行作标题）

## 重新安装

修改配置后，再次双击 `.popclipext` 文件夹即可更新。
